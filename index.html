GWGEN: A global weather generator for daily data¶

tests
package

Welcome! This synthesis of FORTRAN and Python is a globally applicable weather generator inspired by the original WGEN weather generator of [Richardson1981] and parameterized through a global dataset of [GHCN] and [EECRA] data. A paper with the scientific documentation is in progress, this document shall cover the technical documentation.
